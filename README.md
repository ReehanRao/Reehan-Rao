## Featured Projects

### Customer Default Risk Prediction Model
**Industry Capstone | RuralCo NZ, Ashburton | 2025**

> Predicting credit default risk for a New Zealand farming cooperative using 2M+ transactions.

- Engineered features across **6,075 customer accounts** with an extreme class imbalance of 0.18% default rate
- Compared Logistic Regression, Random Forest, and XGBoost with GridSearchCV hyperparameter tuning
- Applied **SHAP explainability** to surface key risk drivers: payment-to-spend ratio, account tenure, and spending volatility
- Segmented customers into **5 behavioural risk profiles** using K-means clustering
- Deployed a **Streamlit dashboard** enabling non-technical staff to inspect model outputs without any coding

**Tech:** Python · scikit-learn · XGBoost · SHAP · pandas · Streamlit · Matplotlib · Plotly

### Spatial Temperature Prediction Across Canterbury
**University of Canterbury | 2025**

> Modelling temperature patterns across Canterbury's complex terrain using 10 years of NIWA climate data.

- Compared **9 spatial interpolation methods** (IDW, Ordinary Kriging, Universal Kriging, Regression Kriging) across ~8,500 prediction locations
- Evaluated using **Leave-One-Out Cross-Validation**
- Regression Kriging achieved best performance (**RMSE ≈ 1.58°C**), a 33% improvement over the next best method
- Identified **elevation as the dominant driver** of temperature variation (~72% explained)

**Tech:** Python · GeoPandas · Kriging · Matplotlib
